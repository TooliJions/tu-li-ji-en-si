**写作页面特殊布局（页面 5/6）：**

写作页面（正文创作、章节阅读与编辑）默认三栏布局，但支持**一键进入心流模式**：
- 顶栏右侧提供 `🌙 心流模式` 按钮
- 点击后隐藏左侧边栏和右侧面板，写作区域全屏居中
- 背景自动暗化，聚焦于正文编辑
- 再次点击或按 `Esc` 退出心流模式

**上下文悬浮气泡（写作页面心流模式专属）：**

- 心流模式下，系统后台自动识别当前屏幕可见区域内的实体词汇（角色名/地点名/道具名）
- 识别到的实体词汇下方显示极弱的虚线底纹（`border-bottom: 1px dashed rgba(255,255,255,0.15)`），不干扰阅读
- 作者只需将光标**悬停**在带虚线底纹的词汇上即可唤出信息卡片，无需选中拖拽
- 信息卡片内容：当前位置、情绪、持有物品、关系（从真相文件实时读取）
- 光标离开或悬停其他词汇，卡片自动切换/消失
- `Ctrl+K` 快捷键手动调出全局搜索（光标处无实体词汇时）
